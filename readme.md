## Execute
Run the file `Firework.exe`.

- KEY_1~5: launch a firework;
- KEY_Q: slow down;
- KEY_E: speed up;
- KEY_ESC: exit;
- Mouse Scroll: zoom;
- Mouse Left: capture/uncapture the cursor;
- Mouse Move: change the direction of sight.
